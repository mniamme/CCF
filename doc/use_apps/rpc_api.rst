RPC API
=======

Application RPCs are exposed under ``/app``. Applications select the authentication policy for each endpoint, and may expose endpoints without authentication.
You can read what application endpoints may look like, as generated by the :doc:`example C++ logging app </build_apps/example_cpp>`, in :doc:`/build_apps/example_rpc_api`.
